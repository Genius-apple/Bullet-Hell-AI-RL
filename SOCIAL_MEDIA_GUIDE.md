# AI 项目社交媒体引流指南

为了让更多人看到你的 Bullet Hell AI 项目，以下是针对小红书、Bilibili 和其他平台的推广策略。

## 1. 核心卖点 (Hook)

在所有平台上，你的核心卖点都是：
*   **视觉冲击**：AI 在密密麻麻的弹幕中毫发无伤的躲避（“神操作”）。
*   **反差感**：AI 刚开始训练时的“笨拙” vs 训练后的“超神”。
*   **技术门槛低**：强调“普通电脑CPU也能跑”、“Python编写”、“开源”。

## 2. 小红书 (Xiaohongshu) 运营策略

小红书用户喜欢**高颜值封面**、**干货教程**和**生活化分享**。

### 封面图 (Cover Image)
*   **样式**：分屏图。左边是 Python 代码（黑底高亮），右边是游戏画面（弹幕最密集的一刻）。
*   **大字标题**：使用醒目的字体（黄色/红色）。
    *   “Python 玩游戏：AI 躲弹幕 99% 胜率！”
    *   “教你用 100 行代码训练一个游戏 AI”
    *   “不仅是玩游戏，更是学 AI（附源码）”

### 标题 (Title)
*   包含关键词：Python, AI, 强化学习, 游戏开发, 毕设, 源码。
*   例子：
    *   “耗时一周，我用 Python 写了个永远不会死的游戏 AI 🤖”
    *   “强化学习入门：手把手教你写一个躲避弹幕的 AI (附 GitHub)”
    *   “大学生毕设推荐：基于强化学习的自动游戏脚本”

### 正文 (Content)
*   **第一段**：直接抛出结果。“训练了 2000 个回合，现在的 AI 已经可以闭着眼睛躲子弹了。”
*   **第二段**：简单科普。“用的是 DQN 算法，不需要显卡，普通笔记本就能跑。”
*   **第三段**：引导互动。“源码已经开源在 GitHub，想学的同学可以**点个关注**，评论区发‘求源码’，我发你链接！”（或者直接说“链接在个人简介”）
*   **标签 (Tags)**：#python #编程 #人工智能 #强化学习 #游戏开发 #程序员 #黑科技 #github #开源项目

## 3. Bilibili (B站) 视频策略

B站用户更硬核，喜欢**完整的技术分享**和**可视化过程**。

### 视频内容规划
*   **0-10秒 (黄金开头)**：放一段 AI 最精彩的躲避操作，配上激昂的音乐。
*   **10-30秒 (引入)**：介绍项目背景，“为了不玩游戏也能通关，我写了这个 AI...”。
*   **30秒-2分钟 (原理解析)**：配合 `train.py` 的可视化界面，解释 AI 是怎么“看到”子弹的（画出雷达图/线条），怎么获得奖励的。
*   **2分钟-结尾 (教程)**：展示如何下载代码、安装环境、运行 `python train.py`。
*   **结尾**：求三连，引导去 GitHub Star。

### 标题
*   “【硬核】我写了一个 AI，它不仅能躲弹幕，还能嘲讽玩家？(附源码)”
*   “Python 强化学习实战：从零开始训练弹幕游戏 AI”

## 4. GitHub 自身优化 (SEO)

*   **About 栏**：在 GitHub 仓库右侧的 About 里，一定要填上 Website (可以是你的 B 站视频链接) 和 Topics (Topics 必须包含：`reinforcement-learning`, `pygame`, `dqn`, `ai`, `python`)。
*   **README**：我们已经写得很好了，但可以在开头加上你的社交媒体链接（“Follow me on Xiaohongshu/Bilibili”）。

## 5. 其他平台
*   **知乎**：搜索“如何入门强化学习？”、“有哪些有趣的 Python 项目？”，在回答中贴上你的项目链接和 GIF 演示。
*   **V2EX / 掘金**：发布技术文章，标题侧重于“技术实现细节”。
